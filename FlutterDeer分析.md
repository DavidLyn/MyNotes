## flutter packages get慢解决办法

+ 执行命令

```
open ~/.bash_profile
```

+ 打开文件，在里面添加上

```
# flutter
export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PATH=${PATH}:/Users/yuanzhiying/Library/flutter/bin
```

+ 执行

```
source ~/.bash_profile
```
